## Getting Started

### First-Time Setup

#### 1. Download and Install

**iOS Users:**
1. Open App Store on your iPhone/iPad
2. Search for "Silni" or scan QR code
3. Tap "Get" to download and install
4. Open app and follow setup instructions

**Android Users:**
1. Open Google Play Store on your Android device
2. Search for "Silni" or scan QR code
3. Tap "Install" to download and install
4. Open app and follow setup instructions

**Web Users:**
1. Visit [silni.app](https://silni.app) in your browser
2. Click "Install App" or add to home screen
3. Follow setup instructions

#### 2. Create Account

1. **Email Registration**:
   - Enter your email address
   - Create a strong password (8+ characters)
   - Add your full name
   - Verify your email address

2. **Social Login**:
   - **Google**: Tap "Continue with Google" and sign in
   - **Apple**: Tap "Continue with Apple" and use Face ID/Touch ID

3. **Biometric Setup**:
   - Enable fingerprint or face ID for quick login
   - Set up secure authentication options

#### 3. Initial Profile Setup

1. **Personal Information**:
   - Add your date of birth
   - Select your gender
   - Upload a profile picture

2. **Preferences**:
   - Choose your preferred language
   - Set your timezone
   - Configure notification preferences

### Quick Start Tutorial

When you first open Silni, you'll see a quick tutorial covering:

1. **Welcome Screen**: Introduction to Silni's mission
2. **Family Addition**: How to add your first family members

## Family Management

### Adding Family Members

#### Manual Addition

1. **Start Adding**:
   - Tap the "+" button on the Relatives screen
   - Select "Add Family Member"

2. **Basic Information**:
   - **Full Name**: Enter the person's full name
   - **Relationship**: Select from dropdown (father, mother, brother, etc.)
   - **Gender**: Select male/female
   - **Priority**: Choose contact priority (1=high, 2=medium, 3=low)

3. **Contact Information**:
   - **Phone Number**: Add mobile number with country code
   - **Email**: Add email address if available
   - **Address**: Add physical address
   - **City/Country**: Add location information

4. **Additional Details**:
   - **Date of Birth**: Add birth date for age-based reminders
   - **Notes**: Add personal notes or reminders
   - **Islamic Importance**: Add relationship context (e.g., "والد - father")

5. **Save**: Tap "Save" to add the family member

#### Contact Import

1. **Import from Phone**:
   - Go to Relatives → Import Contacts
   - Grant permission to access your contacts
   - Select contacts to import
   - Map relationship types for imported contacts
   - Review and confirm import

2. **Import from Google**:
   - Select "Import from Google Contacts"
   - Sign in to your Google account
   - Choose which contacts to import
   - Assign relationship types

3. **Bulk Operations**:
   - Select multiple contacts for batch import
   - Set default relationship type for selected contacts
   - Review and confirm before importing

### Managing Family Members

#### Viewing Family List

1. **Family Overview**:
   - See all family members in a scrollable list
   - View relationship type and priority indicators
   - See last contact date and interaction count
   - Filter by relationship type or priority

2. **Search and Filter**:
   - Use search bar to find specific family members
   - Filter by relationship type (parents, siblings, children, etc.)
   - Filter by priority level
   - Sort by name, relationship, or last contact

#### Editing Family Members

1. **Access Edit Mode**:
   - Tap on any family member in the list
   - Tap the edit icon (pencil) in their profile
   - Or long-press and select "Edit"

2. **Update Information**:
   - Modify any field as needed
   - Update relationship priority
   - Add or change contact information
   - Update personal notes and preferences

3. **Save Changes**:
   - Tap "Save" to update the family member's information
   - Changes sync across all your devices

#### Family Tree View

1. **Access Family Tree**:
   - Tap "Family Tree" from the main menu
   - View interactive family relationship diagram
   - Zoom in/out to see different levels of family

2. **Tree Navigation**:
   - Tap on any family member to see their details
   - Expand/collapse branches to focus on specific lines
   - Use search to find specific people in the tree

3. **Tree Features**:
   - **Relationship Lines**: Visual connections between family members
   - **Generation Levels**: See different generations clearly
   - **Interactive Elements**: Tap to view details and interact

## Interaction Tracking

### Recording Interactions

#### Adding New Interactions

1. **Start Logging**:
   - Go to a family member's profile
   - Tap "Log Interaction" button
   - Or tap the "+" button on the Interactions screen

2. **Interaction Details**:
   - **Type**: Choose from call, visit, message, gift, event, other
   - **Date & Time**: Set when the interaction occurred
   - **Duration**: Add duration in minutes (for calls, visits)
   - **Location**: Where the interaction took place
   - **Notes**: Add details about the interaction
   - **Mood**: How you felt during/after the interaction
   - **Photos**: Add photos from the interaction
   - **Rating**: Rate the quality (1-5 stars)

3. **Save Interaction**:
   - Tap "Save" to record the interaction
   - Interaction automatically updates statistics and streaks

#### Quick Interaction Logging

1. **Quick Log Buttons**:
   - Use quick action buttons from home screen
   - "Called Dad" - automatically logs a call
   - "Visited Mom" - logs a visit with current time
   - "Messaged Brother" - logs a message

2. **Voice Logging**:
   - Tap microphone button to use voice commands
   - Say "Called Ahmed yesterday evening"
   - App automatically parses and logs the interaction

#### Viewing Interaction History

1. **Individual History**:
   - Go to family member's profile
   - Scroll through "Recent Interactions" section
   - See detailed history with photos and notes

2. **Global History**:
   - Go to "Interactions" from main menu
   - View all interactions across all family members
   - Filter by date range, interaction type, or family member

3. **Timeline View**:
   - See chronological view of all interactions
   - Scroll through months and years of family connections
   - Tap any interaction to see full details

#### Interaction Analytics

1. **Interaction Patterns**:
   - View frequency charts showing how often you connect
   - See which family members you interact with most
   - Identify patterns in your communication habits

2. **Quality Tracking**:
   - View average interaction ratings over time
   - See which types of interactions are most meaningful
   - Track improvement in relationship quality

## Gamification Features

### Points System

#### Earning Points

1. **Interaction Points**:
   - **Call**: 10 points
   - **Visit**: 15 points
   - **Message**: 5 points
   - **Gift**: 20 points
   - **Event**: 25 points

2. **Bonus Points**:
   - **Streak Bonus**: Extra points for consistent communication
   - **First Contact**: Bonus for reaching out after long gaps
   - **Quality Bonus**: Extra points for meaningful interactions
   - **Friday Bonus**: Double points for Friday interactions

3. **Point Tracking**:
   - View current point total in profile
   - See points earned this week/month
   - Track progress toward next level

### Achievement System

#### Badge Categories

1. **Consistency Badges**:
   - "Weekly Warrior": 7 consecutive days of interactions
   - "Monthly Champion": 30 days of interactions in a month
   - "Family Keeper": 100 total interactions

2. **Relationship Badges**:
   - "Son's Delight": Strong relationship with parents
   - "Brother's Bond": Regular sibling connection
   - "Family Pillar": Supporting multiple family members

3. **Special Badges**:
   - "Ramadan Star": Extra connections during Ramadan
   - "Eid Champion": Reaching out during Eid holidays
   - "Friday Regular": Consistent Friday family contact

#### Level Progression

1. **Level Requirements**:
   - **Level 1**: 0-50 points (Beginner)
   - **Level 2**: 51-200 points (Regular)
   - **Level 3**: 201-500 points (Dedicated)
   - **Level 4**: 501-1000 points (Family Champion)
   - **Level 5**: 1000+ points (Family Legend)

2. **Level Benefits**:
   - Unlock new features at higher levels
   - Get special badges and recognition
   - Access premium content and insights
   - Earn bragging rights in leaderboards

### Streak System

#### Building Streaks

1. **Daily Streak**:
   - Interact with at least one family member daily
   - Streak continues as long as you maintain daily contact
   - Miss a day? Streak resets to zero

2. **Weekly Streak**:
   - Contact family members at least 5 days per week
   - More flexible than daily streak
   - Partial weeks still contribute to overall consistency

3. **Streak Freezes**:
   - Use "Streak Freeze" items to preserve streaks
   - Earn freeze items through regular activity
   - Strategic use for vacations or busy periods

#### Leaderboards

1. **Family Leaderboard**:
   - Compare with family members who also use Silni
   - Weekly, monthly, and all-time rankings
   - Friendly competition to encourage family connection

2. **Global Leaderboard**:
   - See top users worldwide (optional participation)
   - Anonymous option for privacy
   - Filter by country or region

## Troubleshooting

### Common Issues

#### Login Problems

1. **Forgot Password**:
   - Tap "Forgot Password" on login screen
   - Enter email address
   - Check email for reset link
   - Create new strong password

2. **Account Locked**:
   - Wait 30 minutes after multiple failed attempts
   - Use "Forgot Password" to reset
   - Contact support if issues persist

3. **Social Login Issues**:
   - Check internet connection
   - Verify app permissions
   - Try manual login as alternative
   - Update app to latest version

#### Sync Issues

1. **Data Not Syncing**:
   - Check internet connection
   - Pull to refresh manually
   - Restart the app
   - Check sync status in settings

2. **Conflicting Data**:
   - Choose which version to keep
   - Use "Resolve Conflicts" feature
   - Contact support for complex conflicts
   - Backup data before resolving

#### Notification Issues

1. **Not Receiving Notifications**:
   - Check notification permissions in device settings
   - Verify app is not in battery optimization mode
   - Check notification settings in app
   - Restart device after enabling notifications

2. **Notification Delays**:
   - Check internet connection stability
   - Verify server status
   - Restart notification service
   - Update app to latest version

### Performance Issues

1. **App Running Slow**:
   - Clear app cache in settings
   - Close other apps running in background
   - Restart device
   - Check for app updates

2. **Battery Drain**:
   - Reduce notification frequency
   - Enable battery saver mode
   - Turn off background refresh
   - Use WiFi instead of cellular when possible
